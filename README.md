🚀 Proyectos Destacados
📊 Análisis de Ventas - Superstore
Análisis completo de un dataset de retail con más de 10,000 registros, aplicando técnicas de limpieza, transformación y visualización de datos.

🔧 Tecnologías: Python (Pandas, NumPy), PostgreSQL, Matplotlib/Seaborn

📌 Logros:

Identificación de productos estrella y patrones de estacionalidad

Análisis de concentración de ventas por región y cliente

Cálculo de crecimiento interanual y porcentajes acumulados

🔗 Repositorio: Superstore Python Analysis

Ejercicio técnico simulando un caso real de entrevista para Data Analyst, trabajando con datos sucios y documentando cada decisión.

🔧 Tecnologías: Python (Pandas), Jupyter Notebooks

📌 Logros:

Limpieza y transformación de datos con justificaciones técnicas

Extracción de insights de negocio a partir de datos limitados

Documentación profesional del proceso en GitHub

🔗 Repositorio: RetailNova Análisis
